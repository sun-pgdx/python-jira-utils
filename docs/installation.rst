============
Installation
============


Stable release
--------------

To install python-jira-utils, run this command in your terminal:

.. code-block:: console

    $ pip install python_jira_utils

This is the preferred method to install python-jira-utils, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for python-jira-utils can be downloaded from the `PGDX code repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git@github.com:sun-pgdx/python-jira-utils.git

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/sun-pgdx/python-jira-utils
